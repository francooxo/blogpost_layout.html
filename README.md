# 📋 Portfolio Landing Page - Day 5

This is the final challenge project for Week 1. It pulls together all the structural layouts built throughout the week into one single home landing page dashboard.

---

## ⭐ Features
* **Unified Layout:** Combines the semantic header menu, about sections, and media grids in one space.
* **Clean Reading Flow:** Sections are cleanly divided using standard main content wrappers.
* **Fully Responsive:** Adapts beautifully across both desktop windows and mobile device viewports.

## 🛠️ Built With
* HTML5
* CSS3
* Formspree API
